# AI Medical Report Analyzer

An NLP-based healthcare project that:

- Classifies medical reports into specialties using Machine Learning
- Generates patient-friendly summaries using an LLM API
- Provides results through a web interface

## Progress

### Completed
- Dataset exploration
- Missing value handling
- Top-15 specialty selection
- Class balancing
- Text preprocessing

### Upcoming
- TF-IDF vectorization
- Model training
- Model evaluation
- LLM integration
- Streamlit deployment
